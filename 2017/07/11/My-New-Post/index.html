- 常见的shell
在我们的工作中最常使用的shell一般为一下几种；
1.bsh (bourne shell);
2.ksh (korn shell);
3.csh (C shell);

一般最常用的shell为 sh ， 大部分的unix 系统都将sh 链接到bsh； IBM将sh链接到 Ksh

    shell 程序支持分支以及循环结构， 可以进行信号的处理；
    shell 程序可以加注释，注释一" # "开头，到行尾结束；

- 怎么检测自己使用的shell是什么shell
1、在执行脚本的时候，如果脚本的第一行第一个字符不是" # "号，则启用bsh执行此脚本；
2、在执行脚本的时候，如果脚本的第一个字符不是" # "号，但是第二个字符不是" ! "号，则启用csh 来执行脚本；
3、 在执行脚本的时候，如果脚本的前两个字符为" #! "号，则后面必须是shell 的全路径，则启用指定的shell执行此脚本；如 #！/url / bin /sh


- shell中常用语法
  --shell 中的case语法如下：
      case word in pattern1 : list1 ;; pattern2 : list2;; .... : ... ;;
      esac;
      ## 如果 word 与pattern1 相匹配 则执行 list1 ；与 pattern2匹配则执行list2 ； 与此类推；

      case 语句以 esac 结尾； 每一个分支都要有";;"号来分割开来；

      case 语句中的 pattern 可以使用以下的结构 ：

          1、“或” 结构 ； 如 " a|b|c";
          2、 [ 范围 ] ； 如 " [0-9] " 匹配单个数字;
          3、 " ? " 匹配任何单个字符；
          4、 " * " 匹配任何 0 个或者 多个字符；
          ###  使用 " * ", 相当于C语言中的default分支。

          以上的结构可以组合使用

  --shell 中的 while 循环语句；

      while command
          do
              list 
          done； ## 循环语句必须以 done 结尾；

          1、执行命令为command ；
          2、如果执行成功则 （ 返回值为 0 ）否则则执行 list命令表；
          3、重复1 到command命令失败（返回非0数）；


  --shell 中的 until 循环语法如下：

      until command 
          do
              list
          done   ## 循环语句必须以done结尾，

          1、执行命令 command 
          2、如果执行失败（返回值非0）则执行list 命令列表，
          3、 重复1， 直到command命令执行成功（ 返回值 0 ）；


  --shell 中的 for 循环 语句语法：

      for name in word1 word2 .... word N    
          do
              list
        done  ## 结尾同样 是以 done 结尾

        1、name 为变量名  word1 - wordn 是由空格来分开的单词列表；
        2、for循环每次执行时变量 name 被列表中的下一个单词；
        for循环常用于处理文件集合；


- shell中的特殊字符：

 * ？ [] '' ""  \  $ , & ()  | ^ <>  换行 空格 tab
